Dieses Dialogfeld wird geöffnet, wenn der Zahlungsprozess initiiert wurde. Dies geschieht entweder durch einen Tap auf "Zahlung senden" auf der Startseite oder einen Tap auf eine Feathercoin-URL im mobilen Browser.

Im ersten Feld muss die Empfängeradresse eingegeben werden. Wenn eine Adresse im Adressbuch gespeichert wurde, wird diese automatisch vervollständigt. Es kann ebenfalls ein QR-Code mit Feathercoin-Adressen oder Zahlungsanforderungen eingescannt werden, indem man den QR-Button in der Aktionsleiste tappt.

Im nächsten Feld wird das verschickbare Feathercoin-Guthaben angezeigt. Falls dieses Guthaben vom angezeigten Guthaben auf der Hauptseite abweicht, sind noch nicht alle Transaktionen bestätigt.

Anschließend kann der zu versendende Betrag und die Transaktionsgebühr eingegeben werden. Momentan wird eine Mindestgebühr von 0.0005 LTC empfohlen, um eine Bearbeitung der Transaktion sicherzustellen.

Eine Transaktion kann auch offline eingegeben werden. Sie wird bei einem Neustart von Feathercoin Wallet ausgeführt. Wenn eine Transaktion so schnell wie möglich ausgeführt werden soll, ist eine gute Verbindung (z.B. über WLAN) vonnöten. Außerdem sollte man sichergehen, daß Feathercoin Wallet mit genügend Peers verbunden ist. Dies kann in der Verbindungsübersicht im linken Teil der Mitteilungsleiste überprüft werden.

Achtung: Es gibt keine Möglichkeit, getätigte Transaktionen zurückzurufen. Die Empfängeradresse sollte deswegen vor Bestätigung der Transaktion unbedingt auf Richtigkeit überprüft werden.
