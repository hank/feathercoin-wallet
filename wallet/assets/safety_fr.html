Quelques mots sur la sécurité de vos Feathercoins:

  • Feathercoins dans le Feathercoin Wallet sont stockés dans le téléphone. Si vous le perdez, vous perdez vos Feathercoins. Comme de l'argent liquide dans votre porte-feuille.
  • Avant de désinstaller Feathercoin Wallet (ou d'effacer les données de l'application), transferez vos Feathercoins dans un autre wallet. Tout feathercoins restant seront perdus.
  • Les transactions sont irréversibles. Si vous envoyez vos Feathercoins dans le vide,il n'y a aucun moyen de les récupérer sauf si vous connaissez le destinataire et qu'il est d'accord pour les renvoyer.
  • Gardez votre téléphone Android en sécurité! N'installez que des applications de confiance. Des applications pirates pourraient essayer de voler votre wallet.
  • Minimisez les risques! Utilisez Android Wallet seulement pour des petits montants pour un usage quotidien, et gardez vos économies en sûreté.
